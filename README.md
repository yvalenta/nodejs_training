### NodeJS Training

----

#### Project Setup

- Initialize a new Node.js project:

```bash
npm init -y
```

- Debug Node Application:
  - it opens the chrome dev tools
```bash
node --inspect-brk index.js
```

- Default test library is jest
```bash
npm install --save-dev jest
```

- Run tests
```bash
npm run test
```

- Debugging tests
```bash
node --inspect-brk node_modules/.bin/jest
```
